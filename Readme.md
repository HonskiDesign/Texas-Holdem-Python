# Texas Hold'em

directory structure

```
texas_holdem/
    main/
	__init__.py
	Poker_Main.py
	Winning_Hands_Logic.py
	Card_Deck.py
	NPC_Logic.py
    bin/
    docs/
    setup.py
    tests/
	TexasHoldem_tests.py
	__init__.py
```

## using the skeleton

1. copy this directory to new location
2. reNAME the directory
3. edit the setup file as needed
4. reName the tests/NAME_tests.py 
5. run `nosetests`
6. good to go!
