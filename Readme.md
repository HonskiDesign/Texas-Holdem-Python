# skeleton of a python project

directory structure

```
skeleton/
    NAME/
	__init__.py
    bin/
    docs/
    setup.py
    tests/
	NAME_tests.py
	__init__.py
```

## using the skeleton

1. copy this directory to new location
2. reNAME the directory
3. edit the setup file as needed
4. reName the tests/NAME_tests.py 
5. run `nosetests`
6. good to go!
